# Supercell - 核心页面线框图

## 1. 主界面 - 办公室全景

**交互要点：**
- 每个工位是一个可点击区域，包含：像素角色 + 电脑 + 工位桌面

- 角色有简单的像素骨骼动画（打字、思考、休息等）
- 点击角色 → 弹出员工卡片
- 点击电脑 → 进入聊天记录
- 底部状态栏显示团队概览
- 场景支持缩放和平移
