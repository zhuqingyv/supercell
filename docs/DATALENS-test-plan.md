## 1. 测试范围

### 1.1 产品概要

DataLens（数据透镜）是桌面端本地数据分析工作台。用户导入 CSV/Excel，通过自然语言提问，AI 生成 SQL 查询并可视化展示结果。数据不离开本机，支持离线使用，初期垂直切入电商运营场景。

### 1.2 MVP 功能范围

| 模块 | 功能 | 优先级 |
|------|------|--------|
| 数据导入 | CSV 文件导入（拖拽/选择）、编码检测、分隔符检测、数据预览 | P0 |
| 自然语言查询 | 用户输入问题 -> AI 生成 SQL -> 执行 -> 返回结果 | P0 |
| SQL 展示 | 显示 AI 生成的 SQL + 自然语言解释 | P0 |
| 可视化图表 | AI 推荐图表类型 + 用户切换（柱状图/折线图/饼图/表格） | P0 |
| 模型管理 | 本地 LLM + 可选云端 API，自动检测网络状态 | P1 |
| 查询历史 | 历史查询列表、重新执行、结果对比 | P1 |
| 电商模板 | 预置电商分析模板（GMV/SKU排行/退货率等） | P1 |
| 数据导出 | 图表导出 PNG/PDF、结果导出 Excel | P2 |

### 1.3 不在 MVP 测试范围

- 多表关联查询（推迟到 V2）
- 多平台（MVP 只做一个平台）
- 高级图表类型（热力图、桑基图等）

### 2.2 SQL 准确率测试

详见 [DATALENS-sql-benchmark.md](./DATALENS-sql-benchmark.md)

**验收标准：**

| 查询难度 | 准确率阈值 | 判定标准 |
|----------|-----------|----------|
| 简单（单表聚合/排序/筛选） | >= 90% | SQL 执行结果与标准答案完全一致 |
| 中等（多条件/日期范围/TOP-N/HAVING） | >= 75% | SQL 执行结果与标准答案一致，允许列名/别名差异 |
| 复杂（跨表关联/嵌套子查询） | >= 60%（参考值） | MVP 不要求，作为基线记录 |

**测试方法：**
1. 使用 benchmark 数据集的 50+ 标准查询
2. 将自然语言问题输入 DataLens
3. 提取 AI 生成的 SQL，在标准数据集上执行
4. 对比执行结果与标准答案
5. 按难度分类统计准确率

**回归策略：** 每次模型更新或 prompt 调整后，跑全量 benchmark。
